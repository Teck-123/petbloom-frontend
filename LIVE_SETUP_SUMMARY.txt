## 3 STEPS TO GET YOUR LIVE WEBSITE WORKING

### STEP 1Ô∏è‚É£: Deploy Backend to Live Server
**Choose ONE option:**

**üü¢ EASIEST: Deploy on Railway (same as your DB)**
1. Go to https://railway.app ‚Üí Dashboard
2. Click "New Service" ‚Üí Connect GitHub repo
3. Select `petbloom-frontend` repository
4. Railway auto-reads your `Dockerfile`
5. Set these env vars:
   - `DATABASE_URL` = (auto-filled by Railway)
   - `JWT_SECRET` = (your secret)
   - `FRONTEND_URL` = `https://petbloom-frontend-five.vercel.app`
6. Click Deploy
7. Copy your backend URL from Railway (looks like: `https://petbloom-xxx.railway.app`)

**OR Deploy on Render:**
- Go to https://render.com
- New Web Service
- Connect your GitHub repo
- Settings: Python 3.8, uvicorn start command
- Deploy and get URL
